# 🧰 Chest

Open source terminal ecosystem — emulator, TUI framework, and Lisp scripting.

## Libraries

| Project | Description |
|---------|-------------|
| [Coffer](https://github.com/chestso/coffer) | Virtual terminal engine: VT parser, grid, scrollback, reflow, sixel |
| [Boba](https://github.com/chestso/boba) | TUI library with Elm Architecture — the Bubbletea of C |
| [Ditty](https://github.com/chestso/ditty) | Embeddable Lisp interpreter, Flare syntax highlighter, and CLI |

## Apps

| Project | Description |
|---------|-------------|
| [Portty](https://github.com/chestso/portty) | Modern terminal emulator built on SDL3 with GPU rendering |
| [Mudlark](https://github.com/chestso/mudlark) | MUD/telnet client with Ditty Lisp scripting |
